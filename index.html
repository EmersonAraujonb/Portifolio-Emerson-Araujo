Habilidades

html

É uma linguagem de marcação utilizada na construção de páginas na Web.

css

É um mecanismo para adicionar estilo a um documento web.

Javascript

É uma linguagem de programação. Juntamente com HTML e CSS, é uma das três principais tecnologias da web.

vuejs

É um framework JavaScript de código-aberto, focado no desenvolvimento de interfaces de usuário e aplicativos de página única.

tailwindcss

É um framework CSS de código aberto para design de páginas da web.

bootstrap

É um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web usando HTML, CSS e JavaScript.

styled component

É uma biblioteca (lib) que utiliza o conceito de CSS-in-JS, ou seja, que nos permite escrever códigos CSS dentro do Javascript.

git

É um sistema de controle de versões distribuído, usado principalmente no desenvolvimento de software.

github

É uma plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git.
